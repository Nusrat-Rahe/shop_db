# Online_Bookstore_Management

# 🚀 Project Title

Online Bookstore Management Sysrtem

# 📌 Overview

An interactive and user-friendly Online Bookstore Management System designed to simplify the way people discover and buy books online. It provides a smooth browsing, searching, and ordering experience with a clean and responsive interface. The system ensures secure operations while making management easy for administrators.

# 🏗️ Tech Stack

Frontend:
The frontend is developed using HTML, CSS, and JavaScript to create a clean, responsive, and user-friendly interface. It ensures smooth navigation, attractive UI design, and compatibility across different devices like mobile, tablet, and desktop.

Backend:
The backend is built using PHP (XAMPP environment), which handles server-side logic such as user authentication, book management, cart operations, and order processing. It also manages secure data flow between the frontend and database using structured logic and session handling.

Database:
MySQL is used for database management, designed with properly structured tables such as users, books, orders, cart, and messages. Relationships between tables are maintained using primary and foreign keys to ensure data consistency, and SQL queries (CRUD operations) are used for data manipulation and retrieval.
